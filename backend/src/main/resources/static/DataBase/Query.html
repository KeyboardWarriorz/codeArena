데이터베이스 Query(질의)는 데이터베이스에서 원하는 데이터를 검색하거나 조작하기 위해 사용되는 명령어 또는 문장입니다. Query는 데이터베이스 시스템에게 원하는 작업을 명시적으로 지시하여 데이터를 조회, 삽입, 수정, 삭제하는 데 사용됩니다.

분류SQL 문법설명
DML (Data Manipulation Language)SELECT, INSERT, UPDATE, DELETE데이터 조작을 위한 명령어로, 데이터를 조회, 삽입, 수정, 삭제하는 데 사용됩니다.
DDL (Data Definition Language)CREATE, ALTER, DROP, TRUNCATE, RENAME데이터베이스 구조를 정의하는 명령어로, 테이블, 인덱스, 뷰 등의 객체를 생성, 수정, 삭제하거나 구조를 변경하는 데 사용됩니다.
DCL (Data Control Language)GRANT, REVOKE, DENY데이터베이스에 대한 사용자 권한과 보안을 관리하는 명령어로, 데이터에 대한 접근 권한을 부여하거나 제거합니다.

▼ 일반적으로 많이 쓰이는 SQL 문법

쿼리 종류문법설명
SELECTSELECT 열 FROM 테이블명 [조건]특정 테이블에서 지정된 열을 검색하여 반환합니다. 조건을 지정하여 원하는 데이터를 필터링할 수 있습니다.
INSERTINSERT INTO 테이블명 VALUES (값)지정된 테이블에 새로운 레코드를 삽입합니다. 각 열에 해당하는 값을 지정하여 삽입할 수 있습니다.
UPDATEUPDATE 테이블명 SET 열=값 [조건]특정 테이블의 레코드를 수정합니다. 지정된 열에 해당하는 값을 변경하고, 조건을 지정하여 수정할 레코드를 선택할 수 있습니다.
DELETEDELETE FROM 테이블명 [조건]특정 테이블에서 레코드를 삭제합니다. 조건을 지정하여 삭제할 레코드를 선택할 수 있습니다.
WHEREWHERE 조건SELECT, UPDATE, DELETE 문에서 조건을 지정하여 데이터를 필터링하거나 수정 또는 삭제할 레코드를 선택합니다.
ORDER BYORDER BY 열 [ASC 또는 DESC]SELECT 문에서 결과를 지정된 열의 값에 따라 정렬합니다. ASC는 오름차순, DESC는 내림차순으로 정렬합니다.
GROUP BYGROUP BY 열SELECT 문에서 결과를 지정된 열의 값에 따라 그룹화합니다. 그룹화된 결과에 대한 집계 함수를 사용할 수 있습니다.
JOINJOIN 테이블명 ON 조건두 개 이상의 테이블을 연결하여 관련된 데이터를 함께 조회합니다. 조인 조건을 지정하여 연결할 테이블 간의 관계를 정의합니다.
DISTINCTSELECT DISTINCT 열 FROM 테이블명SELECT 문에서 중복된 값을 제외하고 고유한 값만 반환합니다.
LIMITSELECT 열 FROM 테이블명 LIMIT 숫자SELECT 문에서 반환되는 결과의 개수를 제한합니다. 숫자에는 반환할 최대 레코드 수를 지정합니다.
HAVINGHAVING 조건GROUP BY 문과 함께 사용되며, 그룹화된 결과에 조건을 지정하여 필터링합니다.
