Query(쿼리)는 데이터베이스에서 데이터를 조회하거나 조작하기 위해 사용되는 요청문입니다. 데이터베이스 쿼리는 SQL(Structured Query Language)을 사용하여 작성되며, 데이터베이스에게 원하는 작업을 명령하는 역할을 합니다.

📒

문법설명
SELECT데이터를 조회하는 명령문입니다.
FROM조회할 데이터가 있는 테이블의 이름을 지정합니다.
WHERE데이터를 필터링하는 조건을 지정합니다.
INSERT데이터를 삽입하는 명령문입니다.
UPDATE데이터를 갱신하는 명령문입니다.
DELETE데이터를 삭제하는 명령문입니다.
JOIN둘 이상의 테이블을 연결하여 데이터를 결합합니다.
GROUP BY데이터를 그룹화하여 집계 함수를 적용합니다.
ORDER BY데이터를 정렬합니다.
LIMIT조회 결과의 개수를 제한합니다.

👩‍🏫

위의 문법을 조합하여 쿼리를 작성하면 데이터베이스에서 원하는 데이터를 조회하거나 조작할 수 있습니다. 쿼리는 데이터베이스 시스템에게 명령을 전달하는 중요한 도구이며, 데이터베이스의 유연한 사용과 데이터 처리를 가능하게 합니다.
