📢

자바스크립트에서 객체는 속성(property)과 메서드(method)로 구성된 데이터 구조입니다.

속성은 객체의 특징이나 상태를 나타내는 값이고, 메서드는 객체가 수행할 수 있는 동작을 정의하는 함수입니다. 객체는 중괄호({})를 사용하여 생성하며, 속성과 메서드는 객체의 멤버로 접근할 수 있습니다.

📒

객체와 메서드 예시

// 객체 생성
        var person = {
            name: 'John',
            age: 30,
            sayHello: function() {
                console.log('Hello!');
            }
        };
    
        // 객체 멤버 접근
        console.log(person.name); // 'John' 출력
        person.sayHello(); // 'Hello!' 출력

클래스(class)

클래스는 객체를 생성하기 위한 설계 도구로, 객체의 구조와 동작을 정의합니다. 클래스는 생성자(constructor) 함수, 속성, 메서드 등을 포함할 수 있습니다. 클래스는 class 키워드를 사용하여 정의하고, new 연산자를 사용하여 객체를 생성할 수 있습니다.

// 클래스 정의
        class Person {
            constructor(name, age) {
                this.name = name;
                this.age = age;
            }
    
            sayHello() {
                console.log('Hello!');
            }
        }
    
        // 객체 생성
        var person = new Person('John', 30);
    
        // 객체 멤버 접근
        console.log(person.name); // 'John' 출력
        person.sayHello(); // 'Hello!' 출력

▲ 위의 코드에서 Person 클래스는 name과 age 속성, sayHello 메서드를 가지고 있습니다. new 연산자를 사용하여 Person 클래스의 인스턴스인 person 객체를 생성하고, 해당 객체의 멤버에 접근할 수 있습니다.

👩‍🏫

자바스크립트에서 객체와 클래스를 사용하면 코드를 구조화하고 재사용성을 높일 수 있습니다. 객체는 다양한 속성과 동작을 가지며, 클래스는 객체를 생성하는 템플릿으로 사용됩니다. 객체와 클래스를 적절히 활용하여 유연하고 확장 가능한 코드를 작성할 수 있습니다.
