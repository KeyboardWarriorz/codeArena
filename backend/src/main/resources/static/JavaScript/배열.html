📢

자바스크립트에서 배열(Array)은 여러 개의 값을 순서대로 저장하는 데이터 구조입니다. 배열은 하나의 변수에 여러 개의 값을 저장하고 접근할 수 있으며, 배열 내 요소의 추가, 제거, 수정, 검색 등 다양한 작업을 수행할 수 있습니다.

📒

배열 선언

배열은 [] 기호를 사용하여 선언합니다. 배열에는 0개 이상의 요소를 포함할 수 있으며, 각 요소는 쉼표로 구분됩니다.

var fruits = ['apple', 'banana', 'orange'];

배열 요소 접근

배열 내 요소에 접근하기 위해 인덱스를 사용합니다. 인덱스는 0부터 시작하며, 배열의 요소에는 해당 인덱스를 사용하여 접근할 수 있습니다.

console.log(fruits[0]); // "apple" 출력

배열 길이

배열의 길이는 length 프로퍼티를 통해 확인할 수 있습니다. length는 배열에 포함된 요소의 수를 나타냅니다.

console.log(fruits.length); // 3 출력

배열 요소 추가 및 수정

배열에 요소를 추가하려면 인덱스를 지정하여 값을 할당합니다. 이미 존재하는 인덱스에 값을 할당하면 해당 요소가 수정됩니다.

fruits[3] = 'grape'; // 인덱스 3에 'grape' 추가
        fruits[1] = 'cherry'; // 인덱스 1의 요소를 'cherry'로 수정

배열 요소 제거

delete 연산자를 사용하여 배열에서 요소를 제거할 수 있습니다. 제거된 요소의 자리는 undefined로 남게 됩니다.

delete fruits[2]; // 인덱스 2의 요소를 제거

배열 메서드

자바스크립트 배열은 다양한 내장 메서드를 제공하여 배열에 대한 다양한 작업을 수행할 수 있습니다. 예를 들어, push(), pop(), shift(), unshift() 등의 메서드를 사용하여 배열에 요소를 추가하거나 제거할 수 있습니다.
