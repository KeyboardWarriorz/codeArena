연산자

연산자설명예시
산술 연산자덧셈, 뺄셈, 곱셈, 나눗셈 등의 수학적 연산을 수행합니다.+, -, *, /, %
할당 연산자변수에 값을 할당하거나 수정하는 연산을 수행합니다.=, +=, -=, *=, /= 등
비교 연산자두 개의 값 또는 변수를 비교하여 참(true) 또는 거짓(false)을 반환합니다.==, ===, !=, !==, >, <, >=, <= 등
논리 연산자여러 개의 조건을 조합하여 논리적 평가를 수행합니다.&& (AND), `
삼항 연산자조건식을 평가하여 참인 경우와 거짓인 경우에 따라 다른 값을 반환합니다.조건식 ? 참인 경우 : 거짓인 경우
타입 연산자특정 값의 데이터 타입을 확인합니다.typeof
멤버십 연산자객체 내에서 특정 속성이나 메서드가 존재하는지 확인합니다.in, instanceof
