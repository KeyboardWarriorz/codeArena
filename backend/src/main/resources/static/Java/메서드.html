자바에서 메서드(Method)는 프로그램의 동작을 수행하는 코드 블록입니다. 메서드는 클래스 내에 정의되며, 특정한 작업을 수행하기 위한 코드를 모아둔 함수와 비슷한 개념입니다.

📒

메서드

메서드 구조

메서드 예시 

public class Calculator {
        public static int add(int a, int b) {
            int sum = a + b;
            return sum;
        }
    
        public static void main(String[] args) {
            int result = add(3, 5);
            System.out.println("덧셈 결과: " + result);
        }
    }

▲ Calculator 클래스 내에 add 메서드를 정의하였습니다. add 메서드는 두 개의 정수를 입력받아 덧셈을 수행하고, 그 결과를 반환합니다. 메서드를 호출하기 위해 main 메서드에서 add(3, 5)와 같이 메서드를 호출하고, 반환된 결과를 변수에 저장하여 출력합니다.

👩‍🏫

메서드를 사용하면 코드를 모듈화하고 재사용성을 높일 수 있습니다. 메서드는 특정한 작업을 수행하는 독립적인 코드 블록으로서, 프로그램의 구조를 개선하고 유지보수를 용이하게 합니다.
