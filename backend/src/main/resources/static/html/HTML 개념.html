HTML은 Hyper Text Markup Language의 약자로, 웹 페이지를 작성하고 구조화하기 위한 언어입니다. HTML은 웹 브라우저에게 문서의 구조, 의미, 표현 방법 등을 알려주는 역할을 합니다.

📒

Hyper Text

Markup Language

HTML 요소

요소(태그)설명예시
<h1>제목을 나타내는 요소<h1>Welcome to My Website</h1>
<p>문단을 나타내는 요소<p>This is a paragraph of text.</p>
<a>링크를 생성하는 요소<a href="https://www.example.com">Visit Example Website</a>
<img>이미지를 표시하는 요소<img src="image.jpg" alt="Image Description">
<ul>와 <li>순서 없는 목록을 생성하는 요소<ul> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ul>
<table>표를 생성하는 요소<table> <tr> <th>Header 1</th> <th>Header 2</th> </tr> <tr> <td>Data 1</td> <td>Data 2</td> </tr> </table>
<form>입력 양식을 생성하는 요소<form action="/submit" method="post"> ... </form>
<input>사용자로부터 데이터를 입력받는 요소<input type="text" name="username">
<div>논리적인 구획을 생성하거나 스타일을 적용하는 요소<div class="container"> ... </div>
<span>특정 부분을 그룹화하거나 스타일을 적용하는 요소<span class="highlight">Text</span>
