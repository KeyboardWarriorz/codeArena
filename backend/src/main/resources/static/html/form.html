HTML <form>은 사용자로부터 정보나 데이터를 서버로 제출하기 위한 영역을 정의하는 태그입니다. <form> 요소에는 여러 속성과 태그를 사용하여 폼의 동작과 구조를 정의할 수 있습니다.

📒

action

method

enctype

<input>

<textarea>

<select>

<button>

<label>

예시

<form action="/submit-form" method="POST">
      <label for="name">이름:</label>
      <input type="text" id="name" name="name" required>
    
      <label for="email">이메일:</label>
      <input type="email" id="email" name="email" required>
    
      <label for="message">메시지:</label>
      <textarea id="message" name="message" rows="4" required></textarea>
    
      <input type="submit" value="제출">
    </form>

▲ 위의 예시에서는 <form> 요소를 사용하여 데이터를 제출할 서버의 URL을 /submit-form으로 설정하였고, HTTP 메서드는 POST로 지정하였습니다. 입력 필드로는 이름, 이메일, 메시지를 입력받는 예시가 포함되어 있습니다. <input> 요소의 name 속성은 해당 입력 값의 이름을 지정하며, required 속성은 필수 입력 필드임을 나타냅니다. <input type="submit"> 요소는 폼을 제출하는 버튼을 생성합니다.
