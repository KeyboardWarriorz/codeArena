form

  • < form > 은 정보(데이터)를 서버에 제출하기 위한 영역
  • < form > 기본 속성
    • action: form을 처리할 서버의 URL
    • method: form을 제출할 때 사용할 HTTP 메서드 (GET 혹은 POST)
    • enctype : method가 post인 경우 데이터의 유형
      • application/x-www-form-urlencoded : 기본값
      • multipart/form-data : 파일 전송시 (input type이 file인 경우)
      • text/plain : HTML5 디버깅 용 (잘 사용되지 않음)

태그/속성설명
<form>폼을 생성하는 가장 바깥쪽 태그입니다.
action폼 데이터가 제출될 URL을 지정합니다.
method폼 데이터를 서버로 전송할 HTTP 메서드를 지정합니다. (GET 또는 POST)
target폼 데이터의 응답이 표시될 타겟 윈도우를 지정합니다. (_blank, _self, _parent, _top, 또는 윈도우/프레임의 이름)
<input>사용자로부터 데이터를 입력받는 요소를 생성합니다.
type<input> 요소의 입력 타입을 지정합니다. (text, password, checkbox, radio, submit, reset, 등)
name폼 데이터의 이름을 지정합니다.
value<input> 요소의 초기값을 지정합니다.
<textarea>여러 줄의 텍스트를 입력받는 입력 요소를 생성합니다.
<select>드롭다운 목록을 생성합니다.
<option>드롭다운 목록의 옵션을 정의합니다.
<button>사용자 정의 버튼을 생성합니다.
<label>폼 요소의 레이블을 정의하며, 사용자가 클릭하면 해당 폼 요소로 포커스가 이동합니다.
for<label> 요소와 연결된 폼 요소의 id를 지정합니다.
<fieldset>관련된 폼 요소들을 그룹화하는 컨테이너를 생성합니다.
<legend><fieldset> 요소의 제목을 정의합니다.
