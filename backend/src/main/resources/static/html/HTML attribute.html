HTML 요소에는 속성(attribute)을 사용하여 태그의 부가적인 정보를 설정할 수 있습니다. 속성은 요소의 시작 태그에 작성되며, 이름과 값이 하나의 쌍으로 구성됩니다. 각 태그마다 사용할 수 있는 특정 속성들이 있으며, 몇 가지 속성은 모든 HTML 요소에서 공통으로 사용할 수 있는 HTML Global Attribute라고 불리는 속성들도 있습니다.

📒

HTML Global Attribute

속성설명
class요소에 대한 하나 이상의 클래스를 지정합니다.
id문서 전체에서 고유한 식별자를 지정합니다.
style요소에 대한 인라인 스타일을 지정합니다.
title요소에 대한 추가 정보를 나타내는 툴팁을 제공합니다.
data-*사용자 정의 데이터 속성을 정의합니다.
lang요소의 언어를 지정합니다.
tabindex요소의 탭 순서를 지정합니다.
accesskey요소에 접근하기 위한 키를 지정합니다.
contenteditable요소의 내용을 편집 가능하게 만듭니다.
hidden요소를 숨깁니다.
draggable요소를 드래그 가능하게 만듭니다.
spellcheck요소의 스펠링 및 문법 검사를 활성화/비활성화합니다.

예시

<div id="myDiv" class="container" data-user="john" style="color: blue;" title="This is a div element">
      This is a div element with some attributes.
    </div>
    
    <input type="text" id="name" name="name" tabindex="1" placeholder="Enter your name">
    
    <a href="https://www.example.com" target="_blank" title="Visit Example">Go to Example</a>
