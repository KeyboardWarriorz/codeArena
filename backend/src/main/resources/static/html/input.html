HTML의 <input> 요소는 사용자로부터 데이터를 입력받는 데 사용되는 태그입니다. 다양한 타입과 속성을 가지며, 데이터 입력을 위한 다양한 형태의 위젯을 제공합니다.

📒

<input> 대표적인 속성

속성설명
name폼 컨트롤에 적용되는 이름을 지정합니다 (이름/값 쌍으로 전송됨)
value폼 컨트롤에 적용되는 값(value)을 지정합니다 (이름/값 쌍으로 전송됨)
required폼 컨트롤이 필수 입력 필드임을 나타냅니다
readonly폼 컨트롤이 읽기 전용이며 수정할 수 없음을 나타냅니다
autofocus폼 컨트롤이 자동으로 포커스를 받아야 함을 나타냅니다
autocomplete폼 컨트롤에 자동 완성 기능을 활성화할지 여부를 나타냅니다
disabled폼 컨트롤이 비활성화되어 사용자가 수정할 수 없도록 합니다

<label> 

<label for="agreement">개인정보 수집에 동의합니다.</label>
    <input type="checkbox" name="agreement" id="agreement">

▲ 위의 예시에서 <label> 요소에는 for 속성이 있으며, 해당 속성의 값으로 연결할 <input> 요소의 id 값을 지정합니다. 이를 통해 <label>을 클릭하면 연결된 <input> 요소가 자동으로 활성화되거나 초점을 받을 수 있습니다. 이는 사용자가 레이블을 클릭함으로써 연결된 입력 요소를 쉽게 조작할 수 있도록 도와줍니다.

text

// 일반 텍스트 입력을 받는 유형입니다. 사용자가 텍스트를 입력할 수 있으며, 특정 형식이나 제약 조건은 없습니다.
    <input type="text" name="username">

password

//입력 시 값이 보이지 않고, 특수 기호(*)로 표시되는 비밀번호를 입력받는 유형입니다.
    <input type="password" name="password">

email 

//이메일 형식에 맞는 값을 입력받는 유형입니다. 입력 값이 이메일 형식에 부합하지 않으면 폼을 제출할 수 없습니다.
    <input type="email" name="email">

number

//숫자 값을 입력받는 유형입니다. min, max, step 속성을 사용하여 숫자의 범위와 간격을 지정할 수 있습니다.
    <input type="number" name="quantity" min="1" max="10" step="1">

file

//파일 업로드를 위한 유형입니다. accept 속성을 사용하여 허용할 파일 타입을 지정할 수 있습니다.
    <input type="file" name="file" accept="image/*">

Checkbox

<input type="checkbox" name="fruit" value="apple">

    <input type="radio" name="color" value="blue">

    <input type="radio" name="color" value="green">

Color

<input type="color" name="colorPicker">

Date

<input type="date" name="datePicker">

Hidden

<input type="hidden" name="secretValue" value="12345">

정리 및 그 외 요소

태그/속성설명
<input>사용자로부터 데이터를 입력받는 요소를 생성합니다.
type입력 요소의 타입을 지정합니다. (text, password, checkbox, radio, file, date, 등)
name입력 요소의 이름을 지정합니다.
value입력 요소의 초기값을 지정합니다.
placeholder입력 요소에 힌트 또는 예시 값을 제공합니다.
required입력 요소가 필수 입력인지를 나타냅니다.
disabled입력 요소를 비활성화합니다.
readonly입력 요소를 읽기 전용으로 지정합니다.
checked체크박스나 라디오 버튼의 기본 선택 상태를 지정합니다.
maxlength입력 요소에 입력 가능한 최대 문자 수를 제한합니다.
min숫자 입력 요소의 최소값을 지정합니다.
max숫자 입력 요소의 최대값을 지정합니다.
step숫자 입력 요소의 단계 크기를 지정합니다.
pattern입력 요소의 유효성 검사에 사용할 정규 표현식을 지정합니다.
autocomplete입력 요소에 자동 완성 기능을 활성화합니다.
autofocus페이지가 로드될 때 입력 요소에 자동으로 포커스를 줍니다.
