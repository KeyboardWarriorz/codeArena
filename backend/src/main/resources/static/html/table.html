HTML의 <table> 요소는 행과 열로 구성된 표를 생성하는 데 사용됩니다. 테이블은 주로 데이터를 구조화하고 표시하기 위해 사용됩니다.

📒

태그/속성설명예시
<table>표를 정의하는 최상위 요소<table>
<caption>표의 제목 또는 설명을 나타내는 요소<caption>Monthly Sales</caption>
<thead>표의 헤더 부분을 나타내는 요소<thead> <tr> <th>Product</th> <th>Price</th> </tr> </thead>
<tbody>표의 본문 부분을 나타내는 요소<tbody> <tr> <td>Apple</td> <td>$1.99</td> </tr> </tbody>
<tfoot>표의 바닥글 부분을 나타내는 요소<tfoot> <tr> <td colspan="2">Total: $10.99</td> </tr> </tfoot>
<tr>표의 행을 나타내는 요소<tr> <td>Apple</td> <td>$1.99</td> </tr>
<th>표의 헤더 셀을 나타내는 요소<th>Product</th>
<td>표의 데이터 셀을 나타내는 요소<td>$1.99</td>
colspan 속성하나의 셀이 여러 열에 걸쳐 표시되는 개수를 지정하는 속성<td colspan="2">Total</td>
rowspan 속성하나의 셀이 여러 행에 걸쳐 표시되는 개수를 지정하는 속성<td rowspa

예시

<table>
      <caption>월간 판매 현황</caption>
      <thead>
        <tr>
          <th>상품</th>
          <th>가격</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>사과</td>
          <td>1,000원</td>
        </tr>
        <tr>
          <td>오렌지</td>
          <td>1,500원</td>
        </tr>
      </tbody>
      <tfoot>
        <tr>
          <td colspan="2">총 판매액: 2,500원</td>
        </tr>
      </tfoot>
    </table>

▲ 위 표는 헤더, 본문, 바닥글로 구성되어 있으며, 각각의 요소는 적절한 태그로 묶여 있습니다. 행과 열은 <tr>, <th>, <td> 요소로 정의되었으며, colspan 속성을 사용하여 "총 판매액" 셀을 2개의 열에 걸쳐 표시하였습니다.

👩‍🏫

테이블은 데이터를 구조화하고 시각적으로 표시하는 데 유용한 요소입니다. 각 셀에는 텍스트, 이미지, 입력 필드 등 다양한 유형의 콘텐츠를 포함할 수 있습니다. 테이블을 사용할 때 적절한 마크업과 의미 있는 헤더를 제공하여 웹 접근성을 향상시키는 것이 중요합니다.
