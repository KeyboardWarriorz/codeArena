HTML 문서 구조화는 웹 페이지를 의미 있는 블록으로 분할하여 정보의 구조를 명확하게 표현하는 것을 말합니다. 이를 통해 웹 페이지의 구성 요소를 계층적으로 구조화하여 가독성과 유지 보수성을 향상시킬 수 있습니다.

📒

<header> 

//문서의 머리말 영역을 정의합니다. 일반적으로 웹 페이지의 제목, 로고, 네비게이션 메뉴 등을 포함합니
    <header>
        <h1>웹 페이지 제목</h1>
        <nav>
            <!-- 네비게이션 메뉴 -->
        </nav>
    </header>

<nav>

//네비게이션 영역을 정의합니다. 메뉴 항목이나 링크를 포함하여 웹 페이지의 탐색을 도와줍니다.
    <nav>
        <ul>
            <li><a href="#">메뉴 항목 1</a></li>
            <li><a href="#">메뉴 항목 2</a></li>
            <li><a href="#">메뉴 항목 3</a></li>
        </ul>
    </nav>

<main>

//웹 페이지의 주요 콘텐츠를 정의합니다. 페이지의 핵심 내용이 들어갑니다.
    <main>
        <h2>주요 내용</h2>
        <p>본문 내용...</p>
        <p>추가적인 내용...</p>
    </main>

<footer>

//문서의 바닥글 영역을 정의합니다. 보통 연락처 정보, 저작권 정보, 관련 링크 등이 들어갑니다.
    <footer>
        <p>&copy; 2023 웹 페이지. All rights reserved.</p>
        <nav>
            <!-- 관련 링크 -->
        </nav>
    </footer>

태그설명예시
<a>하이퍼링크를 생성하는 태그로, 다른 문서 또는 같은 문서의 다른 위치로 이동할 수 있도록 합니다.<a href="https://www.example.com">링크</a>
<b>텍스트를 굵게 표시하는 태그입니다.<b>굵은 텍스트</b>
<strong>의미적으로 중요한 텍스트를 강조하는 태그입니다.<strong>중요한 내용</strong>
<i>텍스트를 이탤릭체로 표시하는 태그입니다.<i>이탤릭체 텍스트</i>
<em>텍스트를 강조하여 읽는 사람에게 주목을 유도하는 태그입니다.<em>강조할 내용</em>
<br>줄 바꿈 요소로, 텍스트나 요소들을 줄 바꿈하여 표현합니다.첫 번째 줄<br>두 번째 줄
<img>이미지를 삽입하는 태그로, 웹 페이지에 이미지를 표시합니다.<img src="image.jpg" alt="이미지">
<span>특정 부분을 그룹화하여 스타일이나 동작을 적용하는 인라인 요소를 지정합니다.<span style="color: red;">빨간 텍스트</span>
<p>문단(paragraph)을 정의하는 태그로, 텍스트의 단락을 나타냅니다.<p>문단의 내용</p>
<hr>수평선(horizontal rule)을 나타내는 태그입니다.<hr>
<ol>순서가 있는 목록(ordered list)을 생성하는 태그입니다.<pre><ol><li>항목 1</li><li>항목 2</li><li>항목 3</li></ol></pre>
<ul>순서가 없는 목록(unordered list)을 생성하는 태그입니다.<pre><ul><li>항목 1</li><li>항목 2</li><li>항목 3</li></ul></pre>
<pre>서식이 그대로 유지되는 텍스트(preformatted text)를 정의합니다.`<pre>이 태그 안의 텍스트는 그대로 표현됩니다. 공백도
