JSTL(JSP Standard Tag Library)은 JSP에서 자주 사용되는 작업을 간편하게 처리하기 위한 태그 라이브러리로, 변수 출력, 조건문, 반복문, URL 생성 등을 쉽게 구현할 수 있도록 도와줍니다.

📒

JSTL 요소

태그설명예시
<c:out>변수의 값을 출력합니다.<c:out value="${name}" />
<c:set>변수에 값을 설정합니다.<c:set var="count" value="10" />
<c:if>조건에 따라 내용을 실행합니다.<c:if test="${score >= 80}">합격</c:if>
<c:choose>여러 조건 중 하나를 선택하여 실행합니다.<c:choose><c:when test="${age >= 18}">성인</c:when><c:otherwise>미성년자</c:otherwise></c:choose>
<c:forEach>컬렉션을 순회하며 반복 작업을 수행합니다.<c:forEach items="${list}" var="item"><c:out value="${item}" /></c:forEach>
<c:url>URL을 생성하고 매개변수를 추가합니다.<c:url value="page.jsp"><c:param name="id" value="1" /></c:url>
<c:import>외부 리소스를 포함하거나 다른 JSP 페이지를 포함합니다.<c:import url="common.jsp" />

JSTL 사용

0️⃣ JSTL 라이브러리 선언

<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>

1️⃣ JSTL Core 태그 사용

<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
        <!DOCTYPE html>
        <html>
        <head>
            <title>JSTL 예시</title>
        </head>
        <body>
            <c:set var="name" value="John" />
            <c:if test="${name eq 'John'}">
                <p>안녕하세요, ${name}님!</p>
            </c:if>
            <c:forEach var="i" begin="1" end="5">
                <p>반복문 ${i}</p>
            </c:forEach>
        </body>
        </html>

▲ 위의 예시에서 <c:set> 태그를 사용하여 변수 name을 설정하고, <c:if> 태그를 사용하여 변수 name이 'John'인 경우에만 내용을 출력하고 있습니다. 또한, <c:forEach> 태그를 사용하여 1부터 5까지의 숫자를 반복하면서 내용을 출력하고 있습니다.

👩‍🏫

JSTL을 사용하면 자바 코드 없이도 JSP 페이지에서 반복문, 조건문, 변수 설정 등의 기능을 사용할 수 있습니다. JSTL은 JSP 개발을 간편하고 가독성 좋게 만들어주며, 재사용 가능한 태그들을 제공하여 개발 생산성을 향상시킵니다.
