필터(Filter)는 Java 웹 애플리케이션에서 요청과 응답의 처리를 가로채고, 수정하거나 필터링하는 기능을 제공하는 컴포넌트입니다. 웹 애플리케이션의 특정 URL 패턴에 대해 필터를 설정하면, 해당 URL로 들어오는 요청이나 해당 URL로부터 나가는 응답이 필터에 의해 가로채져 사전 혹은 사후에 처리됩니다.

filter의 역할

  • 요청 필터링 및 수정

    필터는 클라이언트로부터 들어오는 요청(request)을 가로채고, 요청의 특정 조건을 확인하거나 수정할 수 있습니다. 예를 들어, 인증과 관련된 필터는 요청이 인증되었는지 확인하고, 인증되지 않은 요청은 거부하거나 로그인 페이지로 리다이렉트할 수 있습니다

  • 응답 필터링 및 수정

    필터는 서버로부터 클라이언트로 보내지는 응답(response)을 가로채고, 응답을 필터링하거나 수정할 수 있습니다. 예를 들어, 데이터 압축 필터는 서버의 응답을 압축하여 네트워크 대역폭을 절약할 수 있습니다.

  • 공통 작업 처리

    필터는 여러 서블릿이나 JSP 페이지에서 공통으로 수행해야 하는 작업을 처리하기 위해 사용될 수 있습니다. 예를 들어, 로깅 필터는 모든 요청과 응답에 대한 로그를 생성하여 애플리케이션의 동작을 추적할 수 있습니다.
