JSP(JavaServer Pages)는 서버 측 스크립팅 언어로, 동적 웹 페이지 생성을 위해 사용됩니다.

JSP는 HTML 코드와 Java 코드를 조합하여 웹 페이지를 동적으로 생성하고 표현할 수 있습니다.

이를 통해 개발자는 사용자의 요청에 따라 데이터 처리와 로직 수행을 할 수 있습니다.

📒

JSP 특징

  • JSP는 웹 애플리케이션 개발에서 주로 사용되며, 서버에서 실행되어 클라이언트에게 동적으로 생성된 HTML을 전달합니다.
  • JSP 파일은 일반적으로 .jsp 확장자를 가지며, 웹 서버에서 실행되어 동적 HTML을 생성합니다.
  • JSP에서는 <% %> 태그를 사용하여 Java 코드를 삽입할 수 있습니다.
  • JSP는 또한 표현 언어(EL)와 JSTL(JSP Standard Tag Library)을 지원하여 데이터의 출력과 다양한 작업을 간편하게 처리할 수 있습니다.
  • JSP는 다른 서블릿 기술과 함께 사용될 수 있으며, 서블릿 컨테이너에서 실행됩니다.
  • JSP 파일은 서블릿으로 변환되어 컴파일되고, 컨테이너에서 실행됩니다.

➡️ 이를 통해 JSP는 웹 개발에서 유연하고 효율적인 동적 웹 페이지 생성을 가능케 합니다.

JSP 스크립트 요소

문법설명
선언문JSP 페이지 내에서 변수, 메서드, 클래스를 선언합니다.
<%! %>예: <%! int count = 0; %>
스크립트릿JSP 페이지에서 Java 코드를 작성합니다.
<% %>예: <% int result = num1 + num2; %>
표현식JSP 페이지에서 변수나 연산 결과를 출력합니다.
<%= %>예: <%= name %>
주석JSP 페이지에서 주석을 작성합니다.
<%-- --%>예: <%-- This is a comment --%>
지시자JSP 페이지의 전체적인 설정 정보를 지정합니다.
<%@ %>예: <%@ page language="java" contentType="text/html" %>
include 지시자다른 JSP 파일을 현재 JSP 페이지에 포함시킵니다.
<%@ include file="include.jsp" %>
지시문JSP 페이지의 특정 작업을 수행합니다.
<%@ %>예: <%@ taglib uri="http://java.sun.com/jsp/jstl/core" prefix="c" %>

JSP 사용 예시

<%@ page language="java" contentType="text/html; charset=UTF-8" %>
        <!DOCTYPE html>
        <html>
        <head>
            <title>JSP 예시</title>
        </head>
        <body>
            <h1>현재 시간: <%= new java.util.Date() %></h1>
        </body>
        </html>
